# Assets

Thư mục này chứa tài nguyên dùng trong README và phần thuyết trình.

- `pipeline.svg`: sơ đồ pipeline xử lý dữ liệu.
- `6x.png`: screenshot bản đồ ở zoom 6.
- `10x.png`: screenshot bản đồ ở zoom 10.
- `14x.png`: screenshot bản đồ ở zoom 14.

## Chụp Screenshot

1. Chạy `python serve_map.py`.
2. Mở `http://localhost:8080/`.
3. Chọn mức zoom thể hiện rõ các layer và panel.
4. Chụp màn hình và lưu vào thư mục này.
5. Thêm đường dẫn ảnh vào `README.md` nếu có screenshot mới.
